    FormatOneDecisionNode(String, String)

    Returns the formatted text for a single decision node. A decision node is created each time there is a PermitDynamic() transition. There will be a transition from the state that has the dynamic transition to the decision node, and transitions from the decision node to each of the destination nodes. If the caller did not specify the possible destination states, there will be no transitions leaving the decision node.
